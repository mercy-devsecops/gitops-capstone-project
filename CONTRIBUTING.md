# Contributing Guide

## Workflow
- All work must be on a feature branch from main
- I Commit frequently with clear messages
- 3PR required for merge

## Pull Request Process
1. I Open PR from feature branch to main
2. I Included Trello ID in title
3. I Assign reviewers if needed
4. I Wait for CI green build before merge

## CI/CD
- My Lint and tests run automatically
- Build blocked if failures occur

## Coding Standards
- PEP8 for Python
- Newline at EOF
- No trailing whitespace

## Failure Handling
- Fix lint issues
- Fix failing tests
- Push again
